## Stan Gilmour

Architect of prevention informatics. CEO of [Oxon Advisory](https://www.oxonadvisory.com)
and lead of OXAi, its informatics division.

My work sits at the intersection of criminology, public health, and human
rights, focused on multi-agency prevention and the data infrastructure that
makes coordinated prevention possible. I work with police forces, health
boards, local authorities, and international bodies including UNODC.

This GitHub presence holds selected open-source artefacts from that work.
The code here is the production layer behind methodological positions
developed in published research, policy work, and consulting practice.
Where licences permit, the underlying methodology is documented in the
repository's notes; where the methodology is itself part of OXAi's
intellectual property, the code is selectively open and the methodology
is described in plain language.

### Current work

**[CAIRN](https://github.com/SGilr/cairn)** — Place-based prevention
informatics for Scottish public services. An open-data tool that produces
on-demand prevention needs assessments for any Scottish school, drawing
on catchment-weighted SIMD, the evidence library of prevention
interventions, and a curated directory of national delivery partners.
Currently in proof-of-concept with active reviewer engagement.
